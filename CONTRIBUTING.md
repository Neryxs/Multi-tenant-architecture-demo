# Contribuir

¡Gracias por tu interés en contribuir! Por favor, sigue estas pautas:

- Usa ramas feature/
- Haz PRs pequeños y enfocados
- Sigue la guía de estilos y convenciones
- Agrega tests y documentación
- Usa issues para bugs y sugerencias

Para dudas, abre un issue o contacta a los codeowners.
